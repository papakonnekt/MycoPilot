# Requirements: Memory Bank Skill v2 legacy design

EARS-validated functional requirements. Every REQ bullet is one line.

## Functional Requirements (EARS)

- **REQ-150** The archived design record shall preserve the v2 motivation and command architecture for historical traceability.
- **REQ-151** The archived design record shall link implementation evidence through done plans rather than active executable tasks.

## Constraints

- All new behavior must preserve existing defaults unless the linked design explicitly states a v5 major-version gate.
- Tests must be written before implementation for deterministic scripts and validators.
