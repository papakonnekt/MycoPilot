## Методологические заметки

- **POSIX-shell testing**: bats-core поддерживает macOS+Linux, но требует bash 4+ (macOS имеет 3.2 по умолчанию). Установить через `brew install bats-core` или Homebrew bash.
- **shellcheck** — статический анализ shell, CI-friendly, catch edge cases типа неквотированных переменных.
- **Fixture-driven tests** > mock-driven: `tests/fixtures/python/pyproject.toml` даёт реальный parseable файл для `detect_stack`.
